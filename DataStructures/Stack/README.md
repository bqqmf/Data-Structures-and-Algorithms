# 스택
스택을 구현하는 방법에는 2가지가 있습니다.
* array
* linked list

Stack.cpp에서는 array로 스택을 구현했습니다.

## 시간 복잡도

|연산|평균 수행시간|최악 수행시간|
|:-:|:---------:|:---------:|
|Push|`O(1)`|`O(1)`|
|Pop|`O(1)`|`O(1)`|


## 사용 예
* 괄호 짝짓기
* 연산자 표기법 변환
* 오큰수 (NGE)
